About dbrad-jacocoverage

dbrad-jacocoverage is a modification of tikione-jacocoverage and was renamed for two reason:

  • • Identifies plug-in as different from other offering.
  • • Significant changes to the origin source code, in essence a different program.

dbrad-jacocoverage has additional features.

  • Coverage setting:
    • For src and/or test Netbeans project by folders and by package.
    • Associated Netbeans projects.

           Associated Netbeans projects are depends-on library projects or in Netbeans terms sub-projects.

  • Excludes setting:
    • User defined exclude package(s).
    • Exclusion of org.junit, org.testng testing frameworks.
  • Report control:
    • user-defined-directory for reports to be stored to, versus default directory.
    • time-stamped directory.
    • Retention of N previous report directories.
  • Help documentation:
    • Help documentation provided.

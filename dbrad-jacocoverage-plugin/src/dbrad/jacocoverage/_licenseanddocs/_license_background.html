dbrad-jacocoverage background

The software, program and documentation is separate from the original base code and has been renamed to avoid confusion between like software.

Context

  • 'dbrad-jacocoverage' has more capability than 'tikione-jacocoverage', the latter was archived on Github May 12th, 2016.
  • The 'dbrad-jacocoverage' software, program and documentation is separate software from the 'tikione-jacocoverage' base code and has been renamed to avoid confusion between the two software "programs".
  • dbrad-jacocoverage is sub-licensed.
    • The original license(s) for 'tikione-jacocoverage' allowed for re-license/sub-license, or to-use as library/JAR usage.
  • Any email and web-page addresses have been changed from the origin base code.

Modifications

The "software" has significantly modified origin source to allow newer capabilites to enhance the functionality, alongside substantial new source which is under the dbrad-jacocoverage plugin license only.

Other considerations

Portions of 'dbrad-jacocoverage' source code is copyright by its original authors and still applies.

Original license applies to original authors code; however, significant new code has been added to enhanced.where the 'dbrad-jacocoverage' license applies.

Where possible, unmodified origin code is identified for its origin license.

License considerations

'dbrad-jacocoverage' source has retained 'tikione-jacocoverage' license information as best as possible. However, these may be sublicensed under the 'dbrad-jacocoverage license'.

jacocoagent
dbrad-jacocoverage uses libraries for the jacocoagent which are licensed by the LGPL and are used as a library executable via ANT.
MIT applies to substantial portions of software as it pertains to interfacing to the jacocoagent. 'dbrad-jacocoverage' has modified code to interact with newer capabilities, and as such is considered a significant change.
